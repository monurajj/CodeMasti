# CodeMasti - Launching Soon

A cost-efficient, future-ready coding education platform for school students from Class 5 to Class 10.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
